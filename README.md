# Simple Booking Manager

A full-stack booking management system built with the MERN stack (MongoDB, Express, React, Node.js) using TypeScript and Tailwind CSS.

## Features

-  Create new bookings with validation
-  View all bookings in a responsive grid
-  Clean, modern UI with Tailwind CSS
-  Type-safe development with TypeScript
-  Real-time form validation
-  Fully responsive design

## Tech Stack

### Backend
- **Node.js** - Runtime environment
- **Express** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM for MongoDB
- **TypeScript** - Type safety

### Frontend
- **React** - UI library
- **React Router** - Navigation
- **Tailwind CSS** - Styling
- **TypeScript
